## Welcome to my dotfiles

These are my personal dotfiles, built for daily usage on OpenSUSE Tumbleweed with a focus on using KDE Plasma.

---

## System Setup

| Component       | Tool               |
|----------------|--------------------|
| OS             | OpenSUSE Tumbleweed |
| Shell          | Zsh + Zinit        |
| Editor         | Neovim, Zed        |
| Terminal       | Guake              |
| DE             | KDE Plasma         |
| Theme          | Gruvbox + Breeze   |
